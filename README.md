## Development

⚙️ Setup (Windows / macOS / Linux):

1. Install Node.js LTS (v18+ recommended): https://nodejs.org/
2. Install dependencies: `npm install` (or `pnpm install` / `yarn install` if you prefer)
3. Start dev server: `npm run dev`
